# 🧬 Inheritance

This folder contains Java programs demonstrating **advanced Object-Oriented Programming (OOP) concepts**, focusing on:

- Inheritance (single & multilevel)
- Runtime polymorphism
- Method overriding
- Constructor chaining

These programs model **real-world relationships and dynamic behavior in Java**.
