# ☕ Java Basics

This folder contains introductory Java programs focused on building a strong foundation in programming.
The programs cover **basic input/output**, **arithmetic operations**, **control structures**, and **simple logic-building problems**.

These exercises are ideal for:

- 📘 Beginners learning Java
- 🎓 Lab practice and assignments
- 🧠 Strengthening fundamental problem-solving skills
