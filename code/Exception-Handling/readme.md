# ⚠️ Exception Handling

This folder contains Java programs demonstrating **exception handling techniques** used to manage runtime errors and ensure smooth program execution.

The programs cover:

- Built-in exceptions
- Custom (user-defined) exceptions
- Exception propagation
- Multiple exception handling

---

| S. No. | Problem Statement | File |

| 2. | Write a Java program to **handle division by zero using try-catch-finally**. | [ArithmeticExceptionDemo.java](https://github.com/PrateekRaj8125/Java-Core-Concepts/blob/main/code/Exception-Handling/ArithmeticExceptionDemo.java) |

## 🎯 Concepts Covered

- ✅ Exception handling (`try-catch-finally`)
- ✅ Runtime exceptions (`ArithmeticException`, `ArrayIndexOutOfBoundsException`)
- ✅ Custom exceptions (`extends Exception`)
- ✅ Multiple exception handling (multi-catch)
- ✅ Exception propagation (`throw`, `throws`)
- ✅ Input validation using exceptions
