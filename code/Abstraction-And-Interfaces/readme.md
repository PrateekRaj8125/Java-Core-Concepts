# 🧩 Abstraction & Interfaces

This folder contains Java programs demonstrating **abstraction in Object-Oriented Programming (OOP) using**:\

- Abstract classes
- Interfaces
- Interface inheritance

These programs highlight how Java achieves **data hiding**, **modular design**, and **flexibility**.
