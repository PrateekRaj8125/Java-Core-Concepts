# 🤝 Contributing to Java-Core-Concepts

Thank you for your interest in contributing! 🎉  
This repository is designed as a **complete learning path for Core Java**, and your contributions can make it even better.

---

## 📌 Ways to Contribute

You can contribute in the following ways:

- 🐞 Fix bugs or improve existing programs  
- 💡 Add new Java programs (aligned with folder structure)  
- 📝 Improve documentation (README files, explanations)  
- 🚀 Optimize code and improve efficiency  
- 📚 Add comments or explanations for better understanding

## 🛠 Development Setup

1. Fork the repository
2. Clone your fork:

    ```bash
        git clone https://github.com/PrateekRaj8125/Java-Core-Concepts.git
        cd Java-Core-Concepts
    ```

3. Open in your preferred IDE:
    - IntelliJ IDEA
    - Eclipse
    - VS Code
4. Compile and run programs:

    ```bash
        javac FileName.java
        java FileName
    ```

---

## ✅ Contribution Guidelines

- Follow clean coding practices:
  - Proper indentation
  - Meaningful variable and class names
  - Clear comments where necessary
- Maintain folder structure (Level-wise organization)
- Keep programs simple and educational
- Ensure code compiles and runs correctly

---

## 🐞 Reporting Issues

Use the GitHub Issues tab to report:

- Bugs in programs
- Incorrect outputs
- Missing edge cases

Include:

- Description of the issue
- Steps to reproduce
- Expected vs actual output
